Django: Quick Start

Installation: Step by Step

Run the following commands in your Terminal:

  1. Install python3: brew install python3
  2. Install virtualenv: pip3 install virtualenv
  3. Create a directory for your new project: mkdir new-project && cd new-project
  4. Create a new virtualenv for the new project: virtualenv env
  5. Activate the new virtualenv: source env/bin/activate
  6. Install Django in the fresh env: pip install django
  7. Generate scaffolding for your new Django project: django-admin startproject proj
  8. Navigate to the new Django project directory: cd proj
  9. Create a new Django app: python manage.py startapp tasks
  10. Run the Django development server: python manage.py runserver

Helpful Management Commands

  • python manage.py makemigrations: Generates migrations for newly created Django apps.
  • python manage.py migrate: Applies migrates generated by makemigrations
